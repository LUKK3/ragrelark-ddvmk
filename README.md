ragrelark-ddvmk
===============

A graphical roguelike in progress.

Uses SDL and OpenGL.

There's a Code::Blocks project file for compiling.

__Main controls:__
* Numpad for movement.
* < or >: go up/down stairs
* s or I: inventory
* Q: equip
* U: unequip
* d: drop
* , or G: grab
* e: eat
* C: close door
* ctrl: backwards in inventory
* space: select
